

Trang web hỗ trợ tạo và tải thẻ mời tham dự Sinh nhật 5 tuổi của CLB SOL – Speak Out Loud.

##  Cách sử dụng:
1. Truy cập vào trang web theo đường link bên trên. Sau đó vào index.html
2. Nhập tên khách mời vào ô **"Nhập tên khách mời"**.
3. Bấm nút **"Tải Thẻ Mời"** để tự động tải file ảnh dạng `.png` sắc nét (chuẩn kích thước gửi Messenger).
